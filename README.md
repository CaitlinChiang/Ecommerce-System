# E-commerce System

## Goals

- Client: Components for the client-facing side are fully functional, abstract, and dynamic, such that they can be easily tweaked by their design aspect for constant duplication and replication. This is what the customers will see in order to purchase the various services / products being sold.
- Admin: Components for the admin-facing side are fully functional, abstract, and dynamic, such that it is ready for constant duplication and replication. This is what the business owners will see in order to manage the various services / products and other aspects of the client-facing website. 


## Tech Stack

- MERNG: MongoDB, Express.JS, React.JS + Typescript, Node.JS, and GraphQL
- Styled: Material-UI
- Deployed: Heroku & Netlify
